# Autocomplete-Dropdown-Handling-with-Selenium
This project demonstrates how to automate keyboard interactions with an autocomplete dropdown using Selenium WebDriver in Java. The test focuses on simulating user input and navigation through suggestions

Launches the practice page in Chrome.
Types "Sou" into the autocomplete input field.
Waits briefly to allow suggestions to load.
Simulates pressing the DOWN arrow key twice to navigate through the dropdown options.
Prints the currently selected suggestion value from the input field which will be "South Africa"
